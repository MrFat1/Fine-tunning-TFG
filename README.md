# Fine-Tunning LLMs

### 🧪 LLM Medical Inference Experiments (Synthetic Data)

This repository contains a set of experiments and inference runs using instruction-tuned Large Language Models (LLMs) on synthetic medical text.  
The work is part of a TFG (Trabajo Fin de Grado / Bachelor’s Final Project) and focuses on evaluating model behavior, prompt strategies, and inference consistency in a controlled environment.
